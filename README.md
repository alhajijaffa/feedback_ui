# Feedback UI

A simple feedback application built with **HTML**, **CSS**, and **JavaScript**. It allows users to submit feedback through an API with a clean and responsive interface.

## Features

- Responsive design
- Feedback form
- Client-side validation
- Success and error messages

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- REST API
